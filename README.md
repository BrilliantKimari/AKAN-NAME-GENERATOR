

# AKAN-NAME-GENERATOR

## Description  
**AKAN-NAME-GENERATOR** is a fun web application inspired by Ghanaian culture. In Akan tradition, children are named based on the day of the week they were born. This app lets users input their date of birth and gender to discover their unique Akan name.
